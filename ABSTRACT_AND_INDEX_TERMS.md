Abstract-The exponential growth of digital ecosystem reliance has necessitated more robust and secure paradigms for data management than what utilizing traditional centralized cloud storage architectures can offer. Despite their widespread adoption, these centralized systems introduce critical vulnerabilities, most notably single points of failure, susceptibility to censorship, and significant privacy concerns where users effectively surrender sovereignty over their personal information to third party providers. This project presents a decentralized file storage application designed to mitigate these inherent limitations by utilizing the InterPlanetary File System peer to peer hypermedia protocol. Unlike conventional client server interaction models, the proposed solution distributes file data across a distributed global network of independent nodes, ensuring distinct high availability, fault tolerance, and immutable content addressing. The system architecture seamlessly integrates a responsive user interface with a sophisticated back end middleware that interfaces directly with a local distributed network node. By fundamentally eliminating centralized data repositories, this innovative approach significantly enhances data resilience, optimizes bandwidth usage through content deduplication, and effectively prevents unauthorized data tampering. The research and implementation successfully demonstrate a viable transition towards a more secure, user centric web, strictly validating the practical feasibility of decentralized storage as a superior, resilient alternative for future digital preservation and integrity.

Index Terms-Decentralized Storage, InterPlanetary File System, Content Addressing, Distributed Networks, Data Privacy, Peer to Peer Protocol.
